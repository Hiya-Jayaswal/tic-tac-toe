TIC-TAC-TOE
by Hiya Jayaswal

This is a simple Tic-Tac-Toe game built with React. Players take turns marking spaces in a 3Ã—3 grid, aiming to place three of their marks in a horizontal, vertical, or diagonal row.

Features
-Playable Tic-Tac-Toe game for two players.
-Displays the current player.
-Highlights the winning line in green.
-Displays the winner or if the game is a tie.
-Reset button to start a new game.
